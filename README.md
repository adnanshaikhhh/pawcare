# 🐾 PawCare

**Every pet. Every moment. Every detail.**

A premium pet parent companion app — works as a web app and a mobile app (iOS + Android). Built for multi-pet families, family-shared care, AI-assisted health tracking, and emergency vet finding.

> For end-to-end setup instructions, see **[SETUP.md](./SETUP.md)**.

---

## ✨ Features

- 🐱 **Pet profiles** — unlimited pets, photo, breed, age, weight, neutered status
- 💉 **Medical records** — vet visits, medications (with "Give now" log), vaccinations, deworming, weight chart, mood logs
- 🤖 **AI symptom checker** — describe what you notice, get urgency assessment + care tips (uses OpenAI GPT-4o-mini with a rule-based fallback when no API key is set)
- 🚨 **Emergency vet finder** — location-based search for 24/7 vets (Google Places with OpenStreetMap Overpass fallback)
- 📦 **Inventory tracker** — food, litter, medicine with traffic-light stock status and low-stock alerts
- 👨‍👩‍👧 **Family sharing** — share pets and logs with your partner via 8-character invite code
- ⏰ **Smart reminders** — auto-created when you log a vaccine or deworming, plus manual reminders
- 🔔 **Push notifications** — Expo Push (free) for mobile, Web Push for web
- 📄 **PDF export** — one-tap export of any pet's full history for sharing with sitters or new vets
- 🌐 **Works offline-first** — your data is in your Supabase project, accessible from web and mobile

## 🧱 Architecture

```
pawcare/
├── apps/
│   ├── web/                Next.js 14 (App Router) + TypeScript + Tailwind
│   └── mobile/             Expo SDK 50 + React Native + expo-router
├── packages/
│   └── shared/             Shared TypeScript types, Zod validators, constants
├── supabase/
│   ├── migrations/         5 SQL files: schema, RLS, indexes, triggers, helpers
│   └── functions/          Deno edge functions (send-notifications, process-reminders)
└── SETUP.md                Complete deployment walkthrough
```

**Tech stack:**
- **Frontend:** Next.js 14, React 18, Tailwind, shadcn-style components, Framer Motion, recharts, SWR, zustand
- **Mobile:** Expo SDK 50, expo-router, NativeWind, Supabase JS, Expo Notifications
- **Backend:** Next.js API routes (typed + Zod-validated) → Supabase (PostgreSQL + Auth + Storage + Realtime)
- **AI:** OpenAI GPT-4o-mini (optional) with a deterministic rule-based fallback
- **Push:** Expo Push (free, unlimited) for mobile, Web Push API for web
- **Maps:** Google Places (optional) with OpenStreetMap Overpass fallback (free, no key)

## 🚀 Quick start

```bash
# 1. Install dependencies
cd pawcare
npm install

# 2. Set up Supabase
#    - Create a project at https://supabase.com
#    - Run the 5 SQL migrations in supabase/migrations/ in order
#    - Copy project URL + anon key + service role key

# 3. Configure environment
cp .env.example apps/web/.env.local
# Edit apps/web/.env.local with your Supabase credentials

# 4. Run the web app
cd apps/web
npm run dev
# Open http://localhost:3000

# 5. Run the mobile app (optional)
cd ../mobile
npx expo start
# Scan QR with Expo Go on your phone
```

For the full deployment guide including Vercel, EAS (APK build), push notifications, and family-sharing testing, see **[SETUP.md](./SETUP.md)**.

## 📋 What's free, what's not

| Service | Cost | Notes |
|---|---|---|
| Supabase | Free tier (500MB DB, 1GB storage) | More than enough for 10k pets |
| Vercel (web) | Free tier | Hobby plan if you need custom domain |
| Expo Push | **100% free**, no limits | Same API used by Slack/Discord apps |
| EAS Build | Free tier (30 builds/month) | For APK + iOS TestFlight |
| OpenStreetMap (vet search) | **Free**, no API key | Used when Google Places not configured |
| OpenAI | Pay-as-you-go | $5 free credit on new account; app works without it |
| Google Places | $200/month free credit | Optional — falls back to OSM automatically |

**Total cost to run: $0/month** for a single-family deployment.

## 🔐 Security

- All API routes verify Supabase JWT before processing
- RLS policies enforce: you only see your data or your family group's data
- Zod validates every input on every API route
- Rate-limited: AI checker (10/day/user), emergency search (30/hour/IP)
- File uploads: type-checked + size-limited
- PDFs watermarked with "Generated by PawCare"

See **[SETUP.md → Security](./SETUP.md#security--best-practices)** for the full checklist.

## 📱 Mobile build (APK)

```bash
cd apps/mobile
npm install -g eas-cli
eas login
eas build:configure
eas build -p android --profile preview
# EAS gives you a download link for the .apk
```

For iOS without an Apple Developer account, use **Expo Go**:
1. Install Expo Go from the App Store
2. Run `npx expo start` in `apps/mobile/`
3. Scan the QR code with your iPhone camera
